# What will Drive Economic Growth Across Countries in 2026?

## Motivation
Entering the year 2026, many countries are experiencing the long-term effect from teh external shocks, such as the global pandemic. While some countries recover fast enough, some are still struggling. Public debt is still a serious concern to them. For example, the debt-to-GDP ratio of Japan and Itatly already exceed 130%.
